Module Bogue.Timeout

module Timeout: sig .. end

Delayed actions.

Dependency graph
type t 
val add : int -> (unit -> unit) -> t

add delay action will execute action () (in the main thread) after the delay of delay ms.

  • Returns the Timeout element. Warning: don't expect the delay to be exact at the ms. The precision cannot be better than what the framerate imposes, i.e. usually about 16ms. Note: it is legal for an action to add a new Timeout.
